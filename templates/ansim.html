트러블 슈팅

1. 카카오 Map API 마커 렌더링 성능 저하 문제


문제 상황
- 필터링 시 한 번에 수백 개의 마커가 생성되어 렌더링 속도가 급격히 느려짐.
- 지도 이동(드래그, 줌 조작) 시 성능 저하 및 프레임 드랍 발생.

해결 방법
- 지도 줌아웃(zoom out)의 최대 레벨을 제한하여, 너무 넓은 범위를 한 번에 렌더링하지 않도록 설정.
- 마커 클러스터링(MarkerClusterer) 기능을 도입하여 가까운 거리의 마커들을 하나의 군집으로 묶어 표시.
